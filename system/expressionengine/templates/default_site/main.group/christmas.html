Simply Christmas

Creative Journey Christmas Album

Christmas carols are story songs that we all know. They tell the same story of simply a babe in a manger. The vision behind the record is exactly the name: Simply Christmas. We wanted to create a record that conveys, in a fresh way, the story we all know and all need so desperately. And we wanted to have a blast doing it so the listener would have a blast hearing it!
